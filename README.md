# STM32 Environmental Monitoring System

## 基于 STM32 与 Qt 的嵌入式环境监测系统

## 项目简介

本项目是一个基于 STM32 的嵌入式环境监测系统。

系统通过温湿度传感器采集环境数据，并通过 OLED 显示，同时支持按键设置参数、LED报警以及通过 UART 与 Qt 上位机进行通信。

项目主要用于学习嵌入式系统开发，包括外设驱动、数据通信、设备交互以及上位机开发。

---

## 硬件平台

- STM32 MCU
- SHT30 温湿度传感器
- OLED 显示模块
- W25Qxx Flash存储
- 按键输入
- LED报警模块

---

## 软件环境

- STM32CubeMX
- CMake 工程
- C语言
- Qt / C++

---

## 主要功能

计划实现：

- 温湿度数据采集
- OLED数据显示
- 参数设置与保存
- 阈值报警
- UART通信
- Qt上位机监控
